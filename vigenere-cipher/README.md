### VIGENERE CIPHER

Atbash encryption algorithm in Go.<br>
The algorithm assumes characters being encrypted are part of the English alphabet i.e (A-Z)(a-z).<br>


###### Example

```
Using 'lemon' as the key
Plain text is 'attack'

```

| Plain text  | Cipher text |
| ----------- |:-----------:|
|      a      |        l    |
|      t      |        x    |
|      t      |        f    |
|      a      |        o    |
|      c      |        p    |
|      k      |        v    |
